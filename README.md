##Credit Card Fraud Detection

This project implements a Credit Card Fraud Detection system using Machine Learning to identify fraudulent transactions from historical data.

** Technologies Used
Python
Pandas, NumPy
Scikit-learn
Matplotlib, Seaborn
Jupyter Notebook

**Model
Random Forest Classifier

**Features
Data preprocessing and scaling
Handling imbalanced data
Model training and evaluation
Performance metrics (Precision, Recall, F1-score, ROC-AUC)
